# Images Directory

This directory contains screenshots and visual documentation for the Healthcare AI Assistant project.

## Required Images

Please add the following images to this directory:

1. **main-interface.png** - Main application interface showing document management, Google Drive integration, and chat functionality
2. **document-preview.png** - Document preview interface showing uploaded documents with page navigation
3. **chat-interface.png** - Split-screen interface showing document preview on the left and AI chat conversation on the right with source citations
4. **source-highlighting.png** - Document page with highlighted sections showing exact source locations for chat responses
5. **report-generator.png** - Report generation interface allowing users to select documents, choose report sections, and customize report content
6. **document-management.png** - Document management interface showing uploaded files and Google Drive integration

## Image Specifications

- Format: PNG
- Recommended resolution: 1920x1080 or higher
- File size: Keep under 2MB per image for optimal GitHub rendering
